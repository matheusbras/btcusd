# Btcusd

## Usage

First argument is the date
Second argument is the hour

```
$./bin/btcusd 15/10/2020 11

{
              :btc_usd => "11399,11",
       :btc_brl_compra => "64024,22",
        :btc_brl_venda => "64031,06",
    :dolar_ptax_compra => "5,62",
     :dolar_ptax_venda => "5,62"
}

```
